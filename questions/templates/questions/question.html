Number of five digit numbers that can be formed with this condition

How many five digit numbers can be made having the digits 1,2,3 each of which can be used at most thrice in a number?

Tags:

linear-algebra matrices

Unfortunately, you're making distinctions where there aren't any--for example, one digit 1 is the same as any other, though it may have a different meaning once we choose a place for it. The first thing I notice about this problem is that, if I want to directly count the desired numbers, there will be several cases and subcases. We could start with the case of no 1s (with subcases two 2s and three 2s), then the case of a single 1 (with subcases one 2, two 2s, and three 2s), and so on. That seems like a pain, though.
